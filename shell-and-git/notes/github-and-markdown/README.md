# Github and Markdown

## Branches

branches are used to work on features without affecting the main branch.

### Useful codes

git branch <branchname> -> creates a new branch
git branch -> lists all branches
git branch -a -> lists all branches, even deleted ones
git branch -d <branchname> -> deletes the branch

git switch <branchname> -> switches to the branch
git switch -c <branchname> -> creates the branch before switching to it

git push -u origin <branchname> -> pushes the branch online, -u is shorthand for --set-upstream

git pull -> get the latest online version to your local repository
